## 📈 指标的上下游关系

```
R2X ─────┬─────────► 数据信息捕捉能力
         │
         └──► R2y ─────────► 分类判别能力
                │
                └──► Q2 ─────► 真实预测能力
                              (包括泛化)
```

**因果关系：**
```
R2X强 ─→ 更好的数据理解 ─→ R2y可能强
R2y强 ─→ 更好的判别能力 ─→ Q2可能强
但不一定：高R2y + 过拟合 ─→ 低Q2
```
